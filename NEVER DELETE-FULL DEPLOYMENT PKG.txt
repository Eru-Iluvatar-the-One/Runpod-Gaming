| # | Severity | Bug | Root Cause | Fix |
|---|----------|-----|------------|-----|
| 1 | **CRITICAL** | `$bash=@"..."@` expanding here-string makes PowerShell interpret ALL `$()` as subexpressions | `PW=$(printf ...)`, `AUTH=$(curl ...)`, `SID=$(echo ...)` are executed as PowerShell, not passed to bash | Switch to literal `@'...'@` + `.Replace()` tokens |
| 2 | **CRITICAL** | Bash variables `$CFG`, `$WS`, `$WP`, `$LOG`, `$AUTH`, `$SID` expand to empty strings in PowerShell | PowerShell expands them before bash ever sees them | Literal here-string preserves `$` for bash |
| 3 | **CRITICAL** | `$LOG` collision — exists as both PS variable (Windows path) and bash variable (`/workspace/gaming-logs`) | Expanding here-string replaces bash `$LOG` with Windows path | Renamed bash var to `$LOGDIR`; literal here-string eliminates collision |
| 4 | **HIGH** | `2>&1` in bash — PowerShell interprets `$1` as empty variable, producing invalid `2>&` | `$1` is a valid PS automatic variable (empty) | Literal here-string |
| 5 | **HIGH** | CRLF line endings — `Set-Content -Encoding Ascii` writes Windows CRLF, bash chokes on `\r` | `set -e\r` causes bash syntax errors | `[IO.File]::WriteAllBytes()` with explicit LF |
| 6 | **MEDIUM** | SCP nesting — copies `Saved Games/` into `/tmp/wis/`, rsync merges into `$WS/` (which IS `Saved Games/`), creating `Saved Games/Saved Games/` | SCP copies the directory, not its contents | Detect nested dir in bash, adjust rsync source |
| 7 | **MEDIUM** | `b2 authorize-account` / `b2 upload-file` — pip3 installs b2 v4 which removed these commands | b2 CLI v3→v4 breaking change | Replace with `rclone copyto` (already configured) |
| 8 | **MEDIUM** | `$_` parse bug in `">> GQL err $tries: $_"` | `$_:` parsed as drive reference | Already fixed with concatenation; using `${tries}` for clarity |

### `gpu-config.py`

```python
#!/usr/bin/env python3
"""Auto-detect GPU and output optimal Unsloth training config."""
import json, subprocess, sys

def get_gpu_info():
    try:
        out = subprocess.check_output([
            "nvidia-smi", "--query-gpu=name,memory.total",
            "--format=csv,noheader,nounits"
        ], text=True).strip()
        name, vram_mb = out.split(",")
        return name.strip(), int(float(vram_mb.strip()))
    except Exception:
        return "unknown", 0

GPU_CONFIGS = {
    # (max_seq_length, batch_size, grad_accum, quant_bits, max_model_params_B)
    "T4":       (2048,  2,  8, 4, 7),
    "L4":       (4096,  4,  4, 4, 13),
    "A10G":     (4096,  4,  4, 4, 13),
    "RTX 3090": (4096,  4,  4, 4, 13),
    "RTX 4090": (4096,  8,  2, 4, 13),
    "A40":      (4096,  8,  2, 4, 30),
    "L40S":     (8192,  8,  2, 4, 30),
    "A6000":    (8192,  8,  2, 4, 30),
    "A100-40":  (8192, 16,  1, 4, 30),
    "A100-80":  (8192, 16,  1, 4, 70),
    "H100":     (8192, 32,  1, 4, 70),
}

name, vram = get_gpu_info()

# Match GPU
matched = None
for key in GPU_CONFIGS:
    if key.lower().replace("-","").replace(" ","") in name.lower().replace("-","").replace(" ",""):
        matched = key
        break

# Fallback by VRAM
if not matched:
    if vram >= 76000:   matched = "A100-80"
    elif vram >= 44000: matched = "L40S"
    elif vram >= 38000: matched = "A100-40"
    elif vram >= 22000: matched = "RTX 4090"
    elif vram >= 14000: matched = "T4"
    else:               matched = "T4"

seq, bs, ga, bits, max_params = GPU_CONFIGS[matched]

config = {
    "gpu_name": name,
    "gpu_vram_mb": vram,
    "matched_profile": matched,
    "max_seq_length": seq,
    "per_device_train_batch_size": bs,
    "gradient_accumulation_steps": ga,
    "quantization_bits": bits,
    "max_model_params_billions": max_params,
    "recommended_models": {
        "7B": ["unsloth/llama-3-8b-bnb-4bit", "unsloth/mistral-7b-v0.3-bnb-4bit"],
        "13B": ["unsloth/llama-2-13b-bnb-4bit"] if max_params >= 13 else [],
        "70B": ["unsloth/llama-3-70b-bnb-4bit"] if max_params >= 70 else [],
    },
    "sample_train_command": (
        f"python3 train.py "
        f"--model unsloth/llama-3-8b-bnb-4bit "
        f"--max_seq_length {seq} "
        f"--per_device_train_batch_size {bs} "
        f"--gradient_accumulation_steps {ga} "
        f"--load_in_4bit"
    )
}

print(json.dumps(config, indent=2))
```

### `templates/create-template.ps1` — RunPod API automation

# Create RunPod templates via GraphQL API
param(
    [Parameter(Mandatory)][string]$TemplateFile,
    [string]$ApiKey = (G "FunFunPod")
)

$t = Get-Content $TemplateFile | ConvertFrom-Json
$envStr = ($t.env | ForEach-Object { "{key:`"$($_.key)`",value:`"$($_.value)`"}" }) -join ","

$body = @{
    query = @"
mutation {
  saveTemplate(input: {
    name: "$($t.name)"
    imageName: "$($t.imageName)"
    containerDiskInGb: $($t.containerDiskInGb)
    volumeInGb: $($t.volumeInGb)
    volumeMountPath: "$($t.volumeMountPath)"
    startSsh: $($t.startSsh.ToString().ToLower())
    env: [$envStr]
    ports: "$($t.ports)"
    dockerArgs: "$($t.dockerArgs)"
    readme: "$($t.readme)"
  }) { id name }
}
"@
} | ConvertTo-Json

$r = Invoke-RestMethod "https://api.runpod.io/graphql?api_key=$ApiKey" `
    -Method POST -ContentType "application/json" -Body $body
Write-Host "Template created: $($r.data.saveTemplate.id)" -ForegroundColor Green

## Critical Bug Pattern: PowerShell Expanding Here-Strings + Bash

**NEVER** use `@"..."@` (double-quoted here-string) to embed bash scripts.
PowerShell interprets `$variable`, `$(command)`, and `$_` inside them.

**ALWAYS** use `@'...'@` (literal here-string) + `.Replace('__TOKEN__', $value)`.

Example:
```powershell
# WRONG — PowerShell expands $HOME, $(whoami), $1
$bash = @"
echo $HOME
result=$(whoami)
awk '{print $1}'
"@

# RIGHT — bash variables preserved, PowerShell values injected via tokens
$bash = @'
echo $HOME
result=$(whoami)
awk '{print $1}'
PASSWORD='__PW__'
'@
$bash = $bash.Replace('__PW__', $actualPassword)
```

**Also**: Write bash scripts with LF line endings, not CRLF:
```powershell
[IO.File]::WriteAllBytes($path,
    [Text.Encoding]::UTF8.GetBytes(($bash -replace "`r`n","`n")))
```
